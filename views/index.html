More info

WTF ?

This tool has been thought for front-end developers who are tired of working with CSS. Here you can quickly convert your old CSS files to fresh new LESS files. As easy as copy/paste. You can also convert files locally by running the Ruby gem (see git repository above).

Credits

Css2Less is based on the work of Sickill (https://github.com/sickill/css2less). It's brought to you by Thomas Pierson (http://thomaspierson.fr/) & Nico Prat (https://twitter.com/#!/nicooprat). This service run on Sinatra and has been built with HAML & LESS (guess what) with the help of the excellent ACE Editor (http://ace.ajax.org/).
